# СтраховаВАНЯ

Интерактивный гид по страхованию для подростков 11–18 лет.

**Сайт:** https://insurance-guide-six.vercel.app/

---

## О проекте

Образовательный веб-сайт, где персонаж по имени Ваня — 16-летний гимнаст — попадает в реальные жизненные ситуации, а его друзья-эксперты объясняют, как страхование работает на практике. Весь контент на русском языке, подача — разговорная, без сложных терминов.

### Что умеет сайт

- **Интерактивные диалоги** — 5 историй в формате переписки (как в мессенджере): разбитый телефон, потеря багажа, перелом на соревнованиях, затопление квартиры, непонятный страховой договор. Сообщения появляются по очереди с анимацией печатания.
- **Персонажи-эксперты** — Джейсон (спорт и НС), Джордан (гаджеты и поездки), Баффет (имущество), Шерлок (договоры и полисы). У каждого своя история и специализация.
- **Путь обучения** — 5 обучающих модулей от основ до словаря терминов, открываются в модальных окнах.
- **Категории страхования** — ОСАГО, КАСКО, ОМС/ДМС, туризм, имущество, НС — с тултипами при наведении и подробными карточками по клику.
- **Квиз** — 15 вопросов, разбитых по категориям (Основы, Автострахование, Медицина и НС, Путешествия, Для подростков). При правильном ответе показывается иллюстрация. На финальном экране — разбивка результатов по категориям.
- **ИИ-чат** — текстовый ассистент на базе YandexGPT Lite, отвечает на вопросы о страховании. Сообщения бота появляются с эффектом печатания.

## Стек

| Слой | Технология |
|---|---|
| Фронтенд | HTML / CSS / Vanilla JS (один файл `index.html`) |
| Бэкенд | Vercel Serverless Function (`api/chat.js`) |
| ИИ-модель | YandexGPT Lite (`yandexgpt-lite`) |
| База знаний | Локальный RAG — `api/knowledge.js`, retrieval по тегам и ключевым словам |
| Деплой | Vercel + GitHub |

Фреймворки не используются. Внешних JS-библиотек нет. Шрифт — Golos Text (Google Fonts).

## Деплой на Vercel

### 1. Получи API-ключ YandexGPT

1. Зайди в [console.yandex.cloud](https://console.yandex.cloud)
2. Создай сервисный аккаунт с ролью `ai.languageModels.user`
3. Выпусти API-ключ

### 2. Задеплой на Vercel

**Через сайт:**
1. [vercel.com](https://vercel.com) → Add New Project → импортируй репозиторий
2. В разделе Environment Variables добавь:
   - `YANDEX_API_KEY` — твой API-ключ YandexGPT
3. Нажми Deploy

**Через CLI:**
```bash
npm i -g vercel
vercel login
vercel --prod
```

### 3. Локальный запуск

```bash
npm i -g vercel
vercel dev
```

Создай `.env.local`:
```
YANDEX_API_KEY=your_key_here
```

Сайт будет доступен на `http://localhost:3000`.

## Как устроен ИИ-чат

Бэкенд (`api/chat.js`) работает как прокси:

1. Принимает историю сообщений от фронтенда
2. Берёт последнее сообщение пользователя и ищет релевантные фрагменты в `knowledge.js` (простой RAG по тегам и ключевым словам)
3. Передаёт найденный контекст в системный промпт YandexGPT Lite
4. Возвращает ответ на фронтенд

API-ключ хранится только в переменных окружения Vercel и никогда не попадает на клиент.
